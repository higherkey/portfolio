TACTILE PRECISION

The first player is always a point of friction in board gaming. *Choose Who* turns that friction into a moment of tactile delight. Using multi-touch tracking and a physics-inspired selection algorithm, it provides an unbiased, high-energy start to any session.

[ MOBILE MULTI-TOUCH INTERFACE SCREENSHOT ]

THE AUDIT

For *Choose Who*, the goal was zero dependencies. I built a custom touch-handling engine that supports up to 10 concurrent inputs, using the **Web Audio API** for sub-millisecond sound triggering to match visual state changes.

  • Zero-framework Vanilla JS for instant load times.
  • CSS Grid for perfectly circular responsive hit-areas.
  • Local persistence for tournament score tracking.
