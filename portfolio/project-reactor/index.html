THE CONCEPT

Project Reactor was born from a desire to test the limits of modern web performance in a competitive multiplayer setting. Using Svelte 5's new runes system, the game achieves sub-16ms state-to-render cycles, ensuring that every tap is recorded with absolute precision.

NEON GLASS

The visual identity relies heavily on glassmorphism and high-contrast neon highlights. Every UI element is built using Vanilla CSS with advanced backdrop-filters and semi-transparent layers to maintain a premium feel across all device sizes.
